Microsoft Projects

Rebranding Project

Search & replace old branding strings to new ones.
  • Executable program that loops through 1,000's of Xml content files and replaces a list of old branding names with new ones.
  • I built the main search and replace module that would sort a list of old branding strings by longest word count and then use a regular expression to see if any text nodes matched any of the elements from that list.
  • I logged all results so that writers and editors could see what got replaced in their content areas.
  • We then checked all updated Xml files back into the new Caps CMS.
  • Built in .NET / C#.

Custom Reports App

Download custom reports on CMS Xml content - Front End
  • Single page application with ASP.NET controls styled with Bootstrap.
  • User can choose report type from 4 different drop down menus.
  • On form submission, the C# code behind will kick off the correct method and gather data from the different SQL tables.
  • Data is written to an Excel file which is then sent back to the client as a downloaded file.
Back End
  • An executable task runs every 2 hours on a VM that parses 1,000's of Xml content files from the Caps CMS.
  • As the program loops through all Xml child nodes, it tests internal, external, image and token links to see if they are broken or not.
  • Results are then written to multiple SQL tables.
  • Built in .NET / C#.
